# Lutris

`sudo pacman -S lutris` — install all optional deps. If needed: `sudo pacman -S vulkan-devel`.

**Battle.net:** use the installer script from the Lutris site and follow its instructions.

**Troubleshooting:** if a game reports Vulkan "not available", check drivers and environment.
